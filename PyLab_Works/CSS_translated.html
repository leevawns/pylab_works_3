PyLab Works Demos   ( june 2009 )
 
  Select Demo, press Enter or F9 or Double-Click to launch demo
 
Starting the demo might take a few seconds. This is caused by the fact that every demo is started as a full blown PyLab_Works, and therefor you can start more than one demo at the same time to compare them.
 

Expand / Collapse expands or collapse the complete tree.
In Application mode
 


aap

 This demo is added for historical reasons, it was the proof of concept. On the other hand in the design mode this demo shows very well the basic possibilities of PyLab_Works. It shows a very simple program to rotate an image, which can be created in just a few minutes.
