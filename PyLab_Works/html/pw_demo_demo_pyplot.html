Demo: PyPlot   (june 2009)

 

This demo shows a generator and a realtime plot, based on pyplot. The "PyPlot Signal" Bricks extends the signals from the array (which are pure array data), with metadata like signalname, line color and linewidth. Also interesting to note that this is a slow plotting library (look at the framerate at the right bottom, which, set at 50 fps, should be no problem at all on this PC).
